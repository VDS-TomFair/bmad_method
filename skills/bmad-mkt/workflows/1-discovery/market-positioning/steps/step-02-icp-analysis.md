# Step 2: ICP Analysis

## Task
Define Ideal Customer Profile using Jobs-to-be-Done (JTBD) evidence:

1. `{project-root}/_bmad-output/marketing-positioning-{date}.md` — Read Context section
2. `{project-root}/_bmad-output/planning-artifacts/product-brief-*.md` — Job-to-be-Done details

## Framework Application

Apply JTBD framework to identify ICP attributes:

- **Functional Job** — What task is the user trying to accomplish?
- **Emotional Job** — What emotional state is the user trying to achieve?
- **Social Job** — What story does the user want to tell others?

## Output

Append to `{project-root}/_bmad-output/marketing-positioning-{date}.md`:

### ICP Section
- **Job Statement** — User hires product to accomplish X
- **3+ ICP Attributes** — Specific, measurable characteristics
  1. Functional: "Needs X to accomplish Y efficiently"
  2. Emotional: "Feels frustrated by Z current solution"
  3. Social: "Wants to be perceived as A by peers"
- **JTBD Evidence** — Quotes from product brief supporting each attribute
