# Free Tool Strategy Workflow Instructions

## Input Context

Read these BMAD configuration files for context:

1. `{project-root}/instructions/01-bmad-config.md` — Project aliases and settings
2. `{project-root}/instructions/02-bmad-state.md` — Current phase and artifacts
3. `{project-root}/_bmad-output/planning-artifacts/product-brief-*.md` — Product context

## Task Overview

Create free tool strategies using frameworks, idea generation, and distribution methods.

## Content Coverage

### Strategy Frameworks
- **Tool Types** — Calculators, generators, checklists, audits
- **Value Proposition** — Lead gen, SEO, brand awareness, education
- **Target Audience** — ICP mapping, pain points, JTBD
- **Success Metrics** — Conversion rate, engagement, virality

### Idea Generation
- **Brainstorming Methods** — SCAMPER, reverse brainstorming
- **Validation Techniques** — Surveys, interviews, landing pages
- **Scoring Framework** — Impact, feasibility, alignment
- **Prioritization Matrix** — High/low impact vs high/low effort

### Implementation Strategies
- **Development Approaches** — In-house, outsourced, no-code
- **Technical Requirements** — Hosting, security, scalability
- **Integration Points** — CRM, email, analytics
- **Resource Planning** — Budget, timeline, team

### Distribution Methods
- **Marketing Channels** — Organic SEO, social, email, paid
- **Content Strategy** — Blog posts, videos, case studies
- **Promotion Tactics** — Guest posts, partnerships, communities
- **Optimization Levers** — CTAs, forms, landing pages

## Workflow Execution

1. Read BMAD config and product brief for context
2. Select strategy frameworks and generate ideas
3. Plan implementation and distribution strategies
4. Create success metrics and monitoring
5. Define analysis procedures and success criteria

## Final Output

Write results to `{project-root}/_bmad-output/free-tool-strategy-{date}.md` containing:

### Free Tool Strategy Section
- **Free tool strategy framework** with examples
- **Tool idea generation and scoring** with matrix
- **Implementation and distribution plan** with steps
- **Success metrics and monitoring** with tools
