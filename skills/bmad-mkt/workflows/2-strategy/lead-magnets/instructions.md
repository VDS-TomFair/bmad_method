# Lead Magnets Workflow Instructions

## Input Context

Read these BMAD configuration files for context:

1. `{project-root}/instructions/01-bmad-config.md` — Project aliases and settings
2. `{project-root}/instructions/02-bmad-state.md` — Current phase and artifacts
3. `{project-root}/_bmad-output/planning-artifacts/product-brief-*.md` — Product context

## Task Overview

Create high-conversion lead magnets aligned with ICP pain points and integrated with email/CMS workflows.

## Content Coverage

### Lead Magnet Types and Formats
- **Ebooks/Guides** — Comprehensive topic deep-dives
- **Checklists/Cheat Sheets** — Quick reference tools
- **Templates** — Ready-to-use frameworks
- **Calculators** — Interactive value demonstrations
- **Webinars/Events** — Live expert engagement
- **Case Studies** — Proof of results
- **Free Trials/Demos** — Product experience

### ICP Alignment and Pain Point Mapping
- **Job-to-be-Done Alignment** — Map magnet to specific JTBD
- **Pain Point Relevance** — Address core ICP challenges
- **Solution Fit** — Demonstrate product value
- **Data Collection Strategy** — Essential vs nice-to-have fields

### Content Creation and Distribution Strategy
- **Content Brief Development** — Objectives, audience, format
- **Design and Copy Guidelines** — Brand consistency, CTAs
- **SEO Optimization** — Keywords, meta descriptions
- **Distribution Channels** — Blog, social, paid ads, email

### Lead Capture and Nurturing Integration
- **Landing Page Strategy** — Minimal fields, clear value
- **Form Optimization** — Progressive profiling, mobile-first
- **Email Sequence Integration** — Welcome, value, nurture
- **CRM Integration Points** — Tags, scoring, assignment

## Workflow Execution

1. Read BMAD config and product brief for context
2. Brainstorm lead magnet ideas with formats
3. Score each idea for ICP alignment
4. Develop content creation and distribution strategy
5. Plan lead capture and nurturing integration

## Final Output

Write results to `{project-root}/_bmad-output/lead-magnets-{date}.md` containing:

### Lead Magnets Section
- **Lead magnet ideas** with formats and topics
- **ICP alignment scoring** for each idea
- **Distribution channels and CTAs** for each
- **Integration points** with email and CRM
