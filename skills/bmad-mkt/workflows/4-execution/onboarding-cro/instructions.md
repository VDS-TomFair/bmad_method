## Task Overview

Optimize onboarding flows using frameworks, user experience, and activation strategies.

## Content Coverage

### Optimization Frameworks
- **Activation Funnel** — First-time use, value realization, habit formation
- **Friction Points** — Loading times, errors, unclear instructions
- **Progressive Onboarding** — Just-in-time education, feature discovery
- **A/B Testing** — Hypothesis-driven experimentation

### User Experience
- **Page Speed** — Core Web Vitals, loading optimization
- **Navigation** — Clear CTAs, exit points, user paths
- **Accessibility** — WCAG compliance, screen readers
- **Personalization** — Dynamic content, tailored messaging

### Activation Strategies
- **Time-to-Value** — Fastest path to core benefit
- **Engagement Loops** — Rewards, feedback, progression
- **Social Proof** — Testimonials, community features
- **Help Systems** — Tooltips, walkthroughs, support access

### Implementation Guide
- **Tool Selection** — Analytics, A/B testing platforms, user session tools
- **Deployment Steps** — Local testing, staging, production rollout
- **Monitoring Tools** — Google Analytics, Hotjar, FullStory
- **Performance Metrics** — Activation rate, retention, completion time

## Workflow Execution

1. Read BMAD config and product brief for context
2. Select optimization frameworks and analyze friction points
3. Design user experience and activation improvements
4. Create A/B testing hypotheses and implementation guide
5. Define performance metrics and monitoring approaches

## Final Output

Write results to `{project-root}/_bmad-output/onboarding-cro-{date}.md` containing:

### Onboarding CRO Section
- **Onboarding flow analysis** with friction points
- **Activation strategy recommendations** with examples
- **A/B testing hypotheses and metrics** with tools
- **Implementation guide and monitoring** with steps
