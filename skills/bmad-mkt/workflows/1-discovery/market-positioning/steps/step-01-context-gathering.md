# Step 1: Context Gathering

## Task
Read BMAD configuration files to gather project context:

1. `{project-root}/instructions/01-bmad-config.md` — Project aliases and settings
2. `{project-root}/instructions/02-bmad-state.md` — Current phase and artifacts
3. `{project-root}/_bmad-output/planning-artifacts/product-brief-*.md` — Product context

## Framework Application

Apply Jobs-to-be-Done (JTBD) framework to understand user context:

- What job is the user hiring this product to do?
- What are the functional, emotional, and social dimensions of that job?
- What existing solutions is the user firing to hire yours?

## Output

Write findings to `{project-root}/_bmad-output/marketing-positioning-{date}.md`:

### Context Section
- Product purpose from product brief
- Current phase from state file
- Project aliases from config
- JTBD job statement
