# Mia's Content & Channel Execution Tips

## 1. SEO Copy Before Publishing

**Always** optimize content for search before publishing:
- Primary keyword in title, H1, first paragraph
- Semantic keywords in subheaders and body
- Meta description with clear CTA
- Internal links to related content

SEO-first writing increases organic reach 3x.

## 2. Paid Ads Before Social Posts

**Run paid promotion** before organic social:
- Use paid to test audience response
- Retarget organic visitors with lookalike audiences
- Track CPA across both channels
- Scale organic based on paid insights

Paid-first strategy reduces wasted creative effort.

## 3. Email Sequence Before Cold Outreach

**Create nurture sequences** before cold emails:
- Send 3-email sequence before 1:1 outreach
- Include case studies and social proof
- Use same messaging framework as cold email
- Track open/click/convert rates

Sequences warm leads 2x better than cold emails.

## 4. Platform-Specific CTAs

**Optimize CTAs per platform**:
- LinkedIn: "Learn how X helped Y achieve Z"
- Twitter: "Quick tip: X solves Y problem"
- Email: "Ready to try X? Here's how"
- Ads: "[Action] + [Benefit] in [Timeframe]"

Generic CTAs underperform by 40%.

## 5. Performance Over Perfection

**Ship fast, optimize later**:
❌ "Perfect copy takes time" — delays ROI
❌ "Industry best practices" — no proof they work
❌ "Everyone wants this" — no JTBD validation

✅ "Test X copy vs Y copy — measure clicks"
✅ "Run A/B test on CTA — optimize for conversions"
✅ "Scale what works — pause what doesn't"

Speed trumps perfection in execution.

**Close every recommendation with executable next steps:**
❌ "We should do X" — vague and unactionable
✅ "Next: Write 3 ad variations for Google/Meta" 
✅ "Next: Optimize landing page for keyword X"

Execution steps, not strategic visions.
