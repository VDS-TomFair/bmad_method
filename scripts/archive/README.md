# Archived Scripts

## Contents

### csv_to_yaml_converter.py

One-time migration tool that converted all BMAD routing CSV files to YAML
format as part of the YAML canonical routing migration (ADR-0010).

- **Used**: During Phase F (YAML migration)
- **Status**: Completed — all routing CSVs have been converted to YAML
- **Reference**: `docs/adr/0010-yaml-canonical-routing.md`
